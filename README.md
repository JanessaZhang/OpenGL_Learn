# OpenGL_Learn
Use opengl to render image and text, and save the rendering results with opencv

可同时渲染图片和文字，并利用opencv将渲染结果保存成RGB图片

系统： ubuntu18.04 + VScode 
库：glfw、glad、opencv、freetype等，均在项目文件中
